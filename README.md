# Czemu-wasze-rodzini-odchodz-i-nie-s-z-wami.
Tylko super komputer z sieciami neuronowymi SkyNet może wskrzesić zmarłego ze zdjęcia i próbki krwi DNA waszego do syntezy ciał z DNA. I próbka głosu jest potrzebna do wskrzeszenia zmarłych. 
Biblie napisał naród pijaków. Super komputery mogą ożywić zmałrego ale to drogie jest. Niedługo ludzie będą się modlić do super komuterów nie do Jezusa czy żydów.
